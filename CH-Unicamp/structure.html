CH-Unicamp Expressive Visemes Database Structure



The CH-Unicamp Expressive Visemes Images Database is composed of a set of images and a spreadsheet CSV (Comma Separated Values) file which carries information about each image in the database.

The images are saved as PNG files without compression and resolution 1920 x 1080 pixels.

The spreadsheet CSV file has the following structure:


The CSV file is organized as follows:

  • no header

  • each row carries the information of an specific image

  • each column corresponds to an specific information, as follows:

  • First Column → <image_filename>

  • Second Column → corresponding emotion

  • Columns 3 to 7 → phonetic context, phones are represented using PRAAT notation

  • Column 3 → second phone at left

  • Column 4 → first phone at left

  • Column 5 → phone represented by the image viseme (center phone, phone of interest)

  • Column 6 → first phone at right

  • Column 7 → second phone at right

  • Column 8 → corresponding Context Dependent Viseme

  • Column 9 → word or phrase corresponding to the context where the image viseme was captured

  • Columns 10 to 65 → pair of x and y coordinates of landmarks manually annotated in the face.
